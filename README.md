# Домашнее задание 2.1 — Flask

REST API для сайта объявлений: создание, получение, редактирование, удаление.

Поля объявления: заголовок, описание, дата создания, владелец.

- POST — создание
- GET — получение по id или списка
- PATCH — редактирование
- DELETE — удаление

Запуск: `pip install -r requirements.txt && python app.py`
